# InnoFlow 创新开发辅助系统

> **📌 项目说明**
> 本项目是专为**科创课程**设计的**创新开发辅助系统低保真模型（Low-fidelity Prototype）**。系统不连接真实后端服务，所有数据均为静态模拟数据，主要用于教学演示、概念验证和用户流程展示。

## 🎯 系统简介

InnoFlow 是一个面向创新团队的辅助系统原型，旨在帮助用户按照科学的创新流程完成从"发现 → 定义 → 开发 → 交付"的全过程创新工作。系统采用现代简洁的设计风格，通过可视化的方式展示创新项目的完整生命周期。

**核心创新点**：系统集成了 **TRIZ（发明问题解决理论）方法论**，通过AI分析帮助用户系统化地识别技术矛盾、推荐创新原理，并提供理想解分析，将复杂的创新理论转化为易用的工具。

## ✨ 核心功能

### 1. 科学创新流程引导
系统将创新过程划分为四个清晰的阶段：
- **🔍 发现阶段** - 问题探索与机会识别，AI 驱动的 Idea 聚类与发散
- **📋 定义阶段** - 需求分析与方案设计，文档管理和任务跟踪
- **⚙️ 开发阶段** - 原型开发与迭代测试，分布式版本控制可视化
- **🚀 交付阶段** - 成果交付与推广应用，交付物管理

### 2. AI Idea 中心（基于TRIZ方法论）⭐ 核心创新
- **TRIZ AI 问题分析**：基于TRIZ方法论的智能问题分析
  - 理想解（IFR）分析：展示理想解决方案的特征
  - 技术矛盾识别：自动识别改善参数与恶化参数之间的矛盾
  - 探索方向推荐：基于TRIZ矛盾矩阵推荐40个创新原理
  - TRIZ原理详情：提供原理说明、应用案例和适用场景
- Canvas 绘制的气泡聚类可视化
- AI 智能聚类算法演示（模拟）
- Idea 详情查看和发散建议生成
- 方向对比功能：支持对比多个探索方向

### 3. 版本控制可视化
- 多分支结构展示
- 版本历史查看和详情展示
- 版本统计信息（提交次数、分支数、贡献者等）

### 4. AI 会议助手
- 文件上传功能（支持拖拽）
- 自动生成会议摘要（模拟）
- 待办事项提取
- 摘要导出功能

### 5. 项目与团队管理
- 项目列表展示和搜索
- 项目进度追踪
- 团队成员展示
- 项目负责人管理

### 6. 工具商店生态
- 工具浏览和搜索
- 工具详情和评价系统
- 个性化推荐算法
- 工具评分和评论功能
- 工具安装和管理
- 开发者上传工具功能

### 7. 管理员后台（可选功能）
- 插件审核与管理
- 定价协商与审核
- 用户与开发者管理
- 财务数据统计
- 系统配置与设置
> 📖 详细说明请参考 [admin/README.md](admin/README.md)

### 8. 通知系统
- 系统通知管理
- 消息分类和筛选
- 未读消息提醒
- 通知标记和清空功能
- 通知中心页面（`notifications.html`）

### 9. 文档生成器
- 自动分析代码和注释
- 生成API文档、README等
- 支持多种文档类型
- 文档预览和导出
- 文档生成器页面（`docgen.html`）

### 10. 订阅管理
- 订阅套餐选择和管理
- 功能限制说明
- 升级提示和引导
- 订阅页面（`subscription.html`）

## 📖 使用教程

### 1. 登录系统
访问 `index.html` 进入登录页面，输入任意用户名和密码登录即可。

### 2. 查看项目列表
登录后进入项目主页（`dashboard.html`），可以看到所有创新项目卡片，每个卡片显示：
- 项目名称
- 当前阶段
- 进度百分比
- 负责人
- 更新时间

### 3. 进入项目详情
点击任意项目卡片，进入项目详情页（`project.html`），可以看到：
- 项目整体进度条
- 四个创新阶段的卡片，每个卡片显示：
  - 阶段进度
  - 阶段状态（已完成/进行中/未开始）
  - 阶段统计数据

### 4. 探索创新阶段

#### 发现阶段（Idea Center）⭐ 核心功能
1. 点击"发现阶段"卡片进入 `idea.html`
2. **使用 TRIZ AI 问题分析**（重点功能）：
   - 在"🔍 AI 问题分析"区域输入问题描述
   - 点击"🤖 AI 分析问题"按钮
   - 查看**理想解（IFR）分析**：紫色渐变卡片展示理想解决方案
   - 查看**冲突点识别**：系统识别的技术矛盾
   - 查看**推荐探索方向**：基于TRIZ原理的解决方案方向
   - 点击"📚 TRIZ"按钮查看TRIZ原理详情（包含案例和应用场景）
   - 点击"探索此方向 →"基于方向生成具体想法
   - 使用"⚖️ 对比"功能对比多个方向
3. 查看 Canvas 绘制的气泡聚类图
4. 点击气泡查看 Idea 详情和 AI 发散建议
5. 点击"🤖 AI 聚类"按钮触发聚类动画效果
6. 拖动气泡调整位置，查看聚类关系

#### 定义阶段
1. 点击"定义阶段"卡片进入 `define.html`
2. 查看需求文档列表
3. 查看任务跟踪和进度信息

#### 开发阶段（版本控制）
1. 点击"开发阶段"卡片进入 `version.html`
2. 查看版本统计信息
3. 选择不同分支查看版本树
4. 点击版本节点查看版本详情

#### 交付阶段
1. 点击"交付阶段"卡片进入 `deliver.html`
2. 查看交付物清单
3. 查看推广计划和应用场景

### 5. 使用 AI 工具
在项目详情页的"扩展工具"区域，可以访问：
- **AI 会议助手**（`tool.html`）
  - 上传会议录音文件（支持拖拽）
  - 点击"生成摘要"查看模拟生成的会议摘要
  - 查看核心要点、待办事项和创新建议
  - 导出摘要或重新上传
- **文档生成器**（`docgen.html`）
  - 自动分析代码和注释
  - 生成API文档、README等各类文档
  - 支持多种文档类型选择
  - 文档预览和导出功能

### 6. 其他功能
- **通知中心**（`notifications.html`）：查看系统通知和消息
- **工具商店**（`toolstore.html`）：浏览、搜索、评分和安装工具
- **开发者中心**（`developer/developer-center.html`）：开发者管理自己的工具和统计信息

## ⚠️ 重要说明

### 低保真模型特性

本项目作为**低保真模型**，具有以下特点：

1. **静态数据** - 所有数据通过 JavaScript 对象模拟，刷新页面后数据会重置
2. **功能演示** - 部分功能仅展示 UI 和交互流程，未实现完整业务逻辑
3. **模拟算法** - AI 相关功能（聚类、摘要生成）为模拟实现，不调用真实 AI 服务
4. **无后端服务** - 不连接任何真实后端 API 或数据库
5. **演示目的** - 主要用于展示系统概念、验证用户流程和界面设计

- ✅ 科创课程教学演示
- ✅ 用户流程验证和测试

### 不适用场景
