# 管理员系统使用说明

## 功能概述

已实现第一阶段核心功能，包括：

### 1. 管理员登录系统
- **登录页面**: `admin/admin-login.html`
- 独立的管理员登录入口
- 验证码保护机制
- 登录状态管理

### 2. 管理员仪表盘
- **仪表盘页面**: `admin/admin-dashboard.html`
- 数据统计概览（待审核插件、待定价协商、新增用户、平台收益）
- 快速操作入口
- 待处理事项列表

### 3. 插件审核管理
- **审核页面**: `admin/admin-plugins.html`
- 插件列表展示（全部/待审核/审核中/已通过/已驳回）
- 插件详情查看
- 审核操作（通过/驳回）
- 定价协商入口

### 4. 定价协商功能
- **定价管理页面**: `admin/admin-pricing.html`
- 定价协商列表
- 开发者定价 vs 管理员建议对比
- 市场参考价格展示
- 协商历史记录
- 发送定价建议功能
- 最多3轮协商机制

### 5. 插件上传功能
- **上传页面**: `developer/upload-plugin.html`
- 完整的插件信息表单
- 文件上传（支持拖拽）
- 定价设置和理由说明
- 功能标签管理
- 草稿保存功能

### 6. 开发者中心
- **开发者中心主页**: `developer/developer-center.html`
- 开发者统计信息（已上传工具、总下载量、评分等）
- 工具管理（查看、编辑、删除已上传的工具）
- 收益统计和数据分析
- 开发者认证和等级系统

## 使用流程

### 管理员审核流程

1. **登录管理后台**
   - 访问 `admin/admin-login.html`
   - 输入管理员账号和密码（演示版：任意账号密码即可）
   - 输入验证码

2. **查看待审核插件**
   - 在仪表盘查看待审核插件数量
   - 进入"插件审核"页面
   - 筛选不同状态的插件

3. **审核插件**
   - 查看插件详情
   - 评估插件功能和定价
   - 选择"通过审核"或"驳回"
   - 如需协商定价，点击"定价协商"

4. **定价协商**
   - 查看开发者定价和管理员建议
   - 参考市场同类产品价格
   - 发送定价建议给开发者
   - 查看协商历史
   - 最多进行3轮协商

### 开发者上传流程

1. **进入上传页面**
   - 在工具商店点击"上传工具"按钮
   - 或直接访问 `developer/upload-plugin.html`

2. **填写插件信息**
   - 基本信息（名称、分类、版本、描述）
   - 添加主要功能点（至少3个）
   - 设置定价和定价理由
   - 上传插件文件和截图

3. **提交审核**
   - 检查信息完整性
   - 点击"提交审核"
   - 等待管理员审核（1-3个工作日）

## 定价标准

### 价格区间建议

- **免费**: ¥0
- **基础工具**: ¥1-49
- **中等工具**: ¥50-149
- **高级工具**: ¥150-299
- **专业工具**: ¥300+

### 定价审核规则

1. **自动审核条件**
   - 价格 ≤ ¥99
   - 认证开发者
   - 开发者评分 ≥ 4.5

2. **人工审核条件**
   - 价格 > ¥299
   - 新开发者
   - 价格变动 > 30%

3. **协商机制**
   - 最多3轮协商
   - 参考市场同类产品价格
   - 考虑功能复杂度和开发成本

## 技术实现

### 文件结构

```
admin/
├── admin-login.html          # 管理员登录页
├── admin-dashboard.html      # 管理员仪表盘
├── admin-plugins.html        # 插件审核管理
├── admin-pricing.html        # 定价协商管理
├── admin-users.html          # 用户管理（占位）
├── admin-finance.html        # 财务管理（占位）
└── admin-settings.html       # 系统设置（占位）

developer/
├── developer-center.html      # 开发者中心主页
└── upload-plugin.html        # 插件上传页面

js/
├── admin.js                  # 管理员功能模块
└── pricing.js                # 定价管理模块
```

### 核心功能模块

- **admin.js**: 管理员权限检查、审核操作、数据管理
- **pricing.js**: 定价规则、市场参考、价格验证、协商建议
